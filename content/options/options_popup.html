PWGen - Einstellungen pro Internetseite vornehmen

Hier kann man pro Internetseite die allgemeingültigen Vorgaben den eigenen Wünschen entsprechend überschreiben.

Bitte die Seite so angeben wie sie im Button-Menu erscheint.
So wird aus [http://www.google.de] z.B. [www.google.de].

Adresse:

PWGen aktiv: "Nein" auswählen, wenn PWGen auf dieser Domain nicht aktiv sein soll.
Benutzername: Bitte den Standard-Benutzernamen für diese Seite hinterlegen.
Lfd. Nummer: Startpunkt für die laufende Nummer dieser Seite.
Hinweis: Wird automatisch um 1 erhöht, wenn für diese Seite ein neues Passwort angefordert wird.
Sonderzeichen: Angeben, ob im Passwort auf dieser Seite Sonderzeichen enthalten sein dürfen.
Passwortlänge: Bitte die Passwortlänge dieser Seite angeben.
Marker setzen: Auf Wunsch wird kein Marker gesetzt, optional kann das Passwort immer sofort eingegeben werden.
Seiten‑Passwort berechnen: Für bestehende Konten kann man hier die PW-Berechnung abschalten und das Passwort vorgeben. Dann sollte man aber im Firefox den Passwort-Speicher abschalten!
Seiten‑Passwort vorgeben: Passwort für bereits bestehende Konten vorgeben...
Wiederholen: ... und bitte wiederholen.
