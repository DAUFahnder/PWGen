PWGen erklärt

Kurz gesagt: PWGen ist ein Passwortgenerator mit integrierter Benutzerverwaltung.

Das Erzeugen eines neuen Passworts ist mit PWGen nur zwei Mausklicks entfernt. Dabei bleibt jederzeit die Möglichkeit erhalten, das vorherige Passwort wieder abzurufen. PWGen speichert sowohl das ursprüngliche Passwort selbstverständlich verschlüsselt ab, als auch alle weiteren Daten die es speichern soll.

PWGen kann den in Firefox integrierten Passwortspeicher vollständig ersetzen. Es speichert auf Wunsch für jeden Benutzer getrennt die Zugangsdaten auf jeder Webseite und ruft sie mit einem einfachen Mausklick wieder ab. Dabei bietet es auf Wunsch die Option, für jeden Benutzer mehrere "Identitäten" abzuspeichern. So kann man sich zum Beispiel in einem Forum sowohl als PersonX als auch als PersonY anmelden, ohne das PWGen die Zugangs-Daten miteinander vermischt.

PWGen ist vielfältig konfigurierbar. Von der Passwortverwaltung über die Datenspeicherung bis hin zum verwendeten Icon auf den Webseiten ist es den eigenen Wünschen anpassbar. Das geht so weit, dass ein Profi die Möglichkeit hat, das gesammte Aussehen von PWGen zu verändern.

Doch trotz der durchaus vielfältigen Optionen bleibt es für den normalen Benutzer leicht zu bedienen. Es werden im normalen Alltag nur die Funktionen sichtbar, welche man auch ausdrücklich haben will. Man wohnt alleine? Dann braucht man keine getrennten Benutzer und kann die Verwaltung entsprechend abgeschaltet lassen. Man benutzt sowieso den Firefox im Auslieferungszustand? Dann braucht man auch das Design nicht anzupassen und die entsprechenden Felder in den Optionen bleiben versteckt.

Wie kann man PWGen dem eigenen Bedarf anpassen

PWGen's Aussehen den eigenen Wünschen anpassen

  • Eigene Grafik hinterlegen:

    Man kann PWGen eine eigene Grafik als Marker übergeben. Dazu einfach den vollständigen base64-Code in das entsprechende Feld im Reiter "Oberfläche" hinterlegen. Den notwendigen Code bekommt man zum Beispiel auf der Seite https://www.base64-image.de/. Der Code muss dabei diesem Format entsprechen: "data:image/png;base64,iVBORw0...KORK5CYII="
    Die Größe des Icons wird von PWGen dem aktuellen Bedarf angepasst, daher sollte die Grafik flexibel genug sein um in verschiedenen Größen eingesetzt zu werden. Am Besten ist eine Vector-Grafik (.svg), aber jedes Grafikformat welches von Firefox unterstützt wird ist möglich.
